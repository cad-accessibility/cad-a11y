# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Instead, report security issues privately to the maintainers with:

- A summary of the issue
- Steps to reproduce
- Potential impact
- Suggested remediation, if available

The team can be reached via create-contact@uw.edu

## Response Expectations

- Initial acknowledgement target: within 5 business days
- Status update target: within 10 business days
- Fix timeline depends on severity and complexity

## Disclosure

We prefer coordinated disclosure. Please allow reasonable time for remediation before public disclosure.
