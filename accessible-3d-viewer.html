Keyboard Shortcuts

Use these keyboard shortcuts from anywhere on the page:

  • D - Go deeper (increase depth by 10%)
  • S - Go shallower (decrease depth by 10%)
  • V - Switch view (cycle through Front/Back/Left/Right/Top/Bottom views)
  • F - Switch to filled/shaded view
  • O - Switch to outline view
  • C - Switch to cut view
  • 0 - Jump to surface (0% depth)
  • 1-9 - Jump to depth percentage (1=10%, 2=20%, ..., 9=90%)
  • A - Get current status (view, depth, mode)
  • ? - Shortcut reminder
  • Escape - Clear focus from current element

Note: These shortcuts are optimized for Braille displays and screen readers. They work globally without needing to focus on specific elements first.
