cad-a11y /ingest test harness

This page stands in for the partner (Kate's) tool. It POSTs an STL to /ingest with a participant's first name, then gives you the returned user_id and a link that opens the model in the simplified /workshop viewer. It is a static file, so just open it in a browser; the server's CORS is open, so it can post from anywhere.

Sent as first_name. Leave blank for an anonymous ingest.

If empty, the bundled sample pyramid (also in examples/sample-model.stl) is sent.
